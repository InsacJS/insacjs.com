# Carpeta `resource`

Contiene los recursos de un determinado módulo.

```txt
API
  ├─ resources
        └─ api
              └─ v1
                    ├─ personas
                          ├─ persona.controller.js
                          ├─ persona.input.js
                          ├─ persona.middleware.js
                          ├─ persona.output.js
                          └─ persona.route.js
```
